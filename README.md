# CMEPDAexam
Repository on the project I made for the exam of the Computing Methods for Experimental Physics and Data Analysis.
The task is to reproduce the measurement of the differential production cross section for $Z^{0}$ in p-p collision at 13 TeV made by CMS with opend data. For the article I refer to: https://cms-results.web.cern.ch/cms-results/public-results/publications/SMP-17-010/index.html.
