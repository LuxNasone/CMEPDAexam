A raw measurement of distribution for variables used to express differential cross-section (Z trasnverse momentum, rapidity and optimized angle), without any unfolding procedures appplied. This function reads input data from a ROOT file, processes the events, and produces three histograms (TH1D) that can be used for further analysis (e.g. unfolding).

Parameters
fname: file path, required to be a .root file;
outname: name for the output file, required to be a .root file. Default : "Repo/outFiles/NotUnfolded.root";
MT: bool that if true enables multithreading with the option ROOTEnableImplicitMT(). Default : true;
mute: bool that if true disables some secondary comments. Default : false.
Returns
a vector of the three histograms (TH1D) saved on the file called outname, meant to be unfolded later on. In order : [0] : Z tranverse momentum [1] : Optimzed angle = tan(($\pi$ - $\Delta_{\phi}$)/2)/cosh($\Delta_{\eta}$/2) [2] : Z rapidity Advised to visualize with a TBrowser.
Note
Requires ROOT framework and RooUnfold.
Warning
Input file must contain expected tree structure.
