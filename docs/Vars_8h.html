Parameters
nNumber of reconstructed particles.
chargeVector of particle charges.
IsoVector of relative isolation values for each particle.
ptVector of transverse momentum (pT).
etaVector of pseudorapidity (η).
Returns
true if the event contains at least two leptons forming a valid Z⁰ candidate, typically requiring:
  • opposite charge pair,
  • transverse momentum above 25 GeV,
  • isolation below 0.15,
  • pseudorapidity within 2.4 and -2.4; false otherwise.
