Determines whether an event is compatible with a reconstructed Z decay into a muon pair based on kinematic and isolation criteria.

Parameters
nNumber of reconstructed particles.
chargeVector of particle charges.
IsoVector of relative isolation values for each particle.
ptVector of transverse momentums.
etaVector of pseudorapidities.
Returns
true if the event contains at least two leptons forming a valid Z candidate, requiring:
  • opposite charge pair,
  • transverse momentum above 25 GeV,
  • isolation below 0.15,
  • pseudorapidity within 2.4 and -2.4; false otherwise.
